// Laravel Echo + Reverb / Pusher
// Presence channel pro sledovani online ucastniku

Echo.join(`estimation.${sessionId}`)
  .here(users => setParticipants(users))
  .joining(user => addParticipant(user))
  .leaving(user => removeParticipant(user))
  .listen('VoteCast', e => updateVoteStatus(e))
  .listen('VotesRevealed', e => showResults(e))
  .listen('RoundConfirmed', e => nextTask(e))
  .listen('SessionCompleted', e => showSummary(e))

  • VotesRevealed (vsechny hlasy)
