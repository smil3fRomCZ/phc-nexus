Autor: Jan Melicherik · Aktualizovano: 31.03.2026 · v sekci: Uvod do projektu

Prehled architektury

PHC Nexus pouziva modulovou architekturu postavenou na Laravel 13 s Inertia.js v2 pro single-page app zazitek.

Klicove moduly

  • Auth — Google SSO, invite flow, onboarding
  • Organization — oddeleni, tymy, tribes, uzivatele
  • Projects — projekty CRUD, clenstvi, role
  • Work — epiky, ukoly, stavove prechody, kanban
  • Approvals — requesty, votes, delegace

Datova vrstva

Pouzivame PostgreSQL 17 s UUIDv7 primarnimi klici. Vsechny modely pouzivaji HasUuidV7 trait.

Dulezite: Kazdy modul ma vlastni adresarovou strukturu — Models, Controllers, Actions, Enums, Routes, Tests.

Frontend stack

React 19 + TypeScript + Tailwind CSS 4 + shadcn/ui. Rich text editor pouziva Tiptap.
