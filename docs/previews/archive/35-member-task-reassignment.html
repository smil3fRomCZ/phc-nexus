💡 UX poznámky:
• Bulk select hromadně nastaví všechny dropdowny — uživatel může pak per úkol upravit
• Checkbox vlevo = označeno k přeřazení (default zapnuto když má vybraného assignee)
• Tlačítko „Potvrdit a odebrat" se uvolní jen když všechny úkoly mají nového assignee
• Po potvrzení: backend provede atomickou transakci — přiřadí všechny úkoly + odebere člena
• Vlastník projektu i odebíraný člen jsou automaticky vyloučeni z dropdownu reassignee
